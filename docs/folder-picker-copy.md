# Folder Picker Copy

## Selected (文艺)

```
每一段思绪都该有一个安静的归处。
选择一个文件夹——Aoroza 将你的文字存为纯粹的 Markdown 文件，
就留在它们本该在的地方：你的电脑上。
```

English:
```
Every thought deserves a quiet home.
Pick a folder — Aoroza will keep your words as simple Markdown files,
right where they belong: on your computer.
```

---

## Alternative A: 诗意

```
文字落地，即为归处。
选一个文件夹，Aoroza 把每一个字写成 .md 文件，
安静地躺在你电脑的角落。
```

English:
```
Where words land, home begins.
Pick a folder. Aoroza writes every word as a .md file,
quietly resting in a corner of your computer.
```

## Alternative B: 温暖

```
你的笔记不需要云端。
它们只需要一个文件夹——在这个电脑上，以 Markdown 的形式，
永远属于你。
```

English:
```
Your notes don't need the cloud.
Just a folder — on this computer, as Markdown,
forever yours.
```

## Alternative C: 极简

```
无云、无账号、无锁定。
选一个文件夹，你的文字以纯文本的形式住在这里。
就这么简单。
```

English:
```
No cloud. No account. No lock-in.
Pick a folder. Your words live here as plain text.
That's it.
```
